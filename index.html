Chrome Dev Tools Setup

This repository contains the information on how to setup the dev tools in Chrome.

View on GitHub Download .zip Download .tar.gz

Step 1 - Download sources of example as zip file

https://github.com/EmbeddedTestingWorkshop/Embedded_FixtureFinder

﻿﻿

Step 2 - Extract to local folder

Step 3 - Open FixtureFinder.html in Chrome

Step 4 - Open Chrome Dev Tools

Go to Sources Panel

Add locally stored files as a folder to the workspace

Allow Chrome to access the local folder

Select FixtureFinder.html, right click, select "Map to File System Resource" -> Select the FixtureFinder.html file from the folder where you saved the downloaded files

Select the FixtureFinder.html file from the folder where you saved the download

Restart Dev Tools by clicking “OK”
